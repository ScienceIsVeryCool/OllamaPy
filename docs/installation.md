# Installation

## Prerequisites

- Python 3.8 or higher
- Ollama server installed and running

## Install OllamaPy

```bash
pip install ollamapy
```

## Start Ollama

Ensure Ollama is running:

```bash
ollama serve
```

## Test Installation

```bash
ollamapy --hello
```

## Configuration

OllamaPy works out of the box with default settings, but can be customized as needed.
