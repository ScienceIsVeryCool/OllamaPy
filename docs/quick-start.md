# Quick Start

Get up and running with OllamaPy in minutes.

## 1. Install OllamaPy

```bash
pip install ollamapy
```

## 2. Start Ollama

```bash
ollama serve
```

## 3. Run OllamaPy

```bash
ollamapy
```

## 4. Try Some Commands

- Ask questions naturally
- The AI will automatically detect when to use skills
- Type `exit` to quit
